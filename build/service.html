如何禁止IIS的自动启动？

（1）win+R 进入运行框，输入Services.msc，然后enter


（2）进入到服务列表，找到 IIS Admin Services，右键，选择属性，在弹出框中的启动类型选择手动启动


（3）进入到服务列表，找到 World Wide Web Publishing Service，右键选择属性，启动类型选择手动

重启电脑，就会发现IIS不会自动启动了


如何进行终端网络服务器连接

   （1）进入“小百合Player”界面，打开系统设置—网络设置，连接有线网络或无线网

        络，连接成功，网络设置显示WiFi图标；


     


   （2）返回“小百合Player”界面，打开服务器设置，出现登录界面弹窗；分为云端登

        录和企业登录，选择云端登录，输入设备名称和机构ID，其他选项默认，单

        击“确认”，连接成功，服务器设置模块右上角图标变为蓝色；或者选择企业登

        录，这时需要输入服务器IP地址“lango-tech.cn”，连接成功显示同上。 


      


   （3）网络和服务器都连接成功，各自所属模块的右上角为蓝色标识；

 

      

   （4）打开后台发布系统，单击菜单栏的终端管理—终端信息，查看是否存在与机构

        名一致的终端，若存在表示连接成功。

如何查找终端登录页面编号

Q：终端登录页面的编号在哪里查找

 

A：在机构管理里面查

 

Q：在机构管理里没找到机构登录编号

 

A：

原因：因为名称太长，导致无法全部显示，编号被隐藏。

解决：进入机构管理把鼠标放在机构名称上，可显示编号

 


为何发布系统无法远程控制终端

可能发布系统未连接终端或终端网络信号差。首先请查看“服务器设置”和“网络设置”的图标是显示白色还是蓝色，白色为未连接，蓝色为连接成功。如下图为成功连接网络和服务器：

 

若终端是用无线连上网的，可能是无线信号差导致后台无法远程控制终端，可以尝试让终端连接有线网络。


如何设置终端定时开关机

1.单击菜单栏的终端管理-终端设置，进入终端设置页面，选择“定时开关机”选项；

2.从机构列表勾选机构，再勾选需要设置的终端；

3.设置星期模式，若其他星期也需要设置，点击“应用到其他星期”，勾选具体星期；

4.选择设置模式，若选择“常开”，无时间段设置，一天24小时保持开机状态；

5.若选择“定时”，打开开关，可以设置四个时间段，注意时间段不可交叉重叠；

6.最后点击“保存设置”，定时开关机设置完毕，数据同步更新到终端。


 


或者在终端进行设置：

1.进入APK界面，单击系统设置—分时设置—开关机分时设置；

2.选择设置模式，若选择“每天模式”，打开时间段控制开关，填写时间段，最多设置四个时间段，注意不可交叉重叠；

3.点击“立即生效”，终端每天都按照设置的时间段进行开关机，自动关机前出现一分钟倒计时提示；

4.若选择“星期模式”，先设置开关机控制模式，分为常关、常开和定时；接着选择星期X，设置开关机时间段，最多设置四个时间段，注意不要交叉重叠；

5.点击“立即生效”，定时开关机设置成功。
