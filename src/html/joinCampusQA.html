你问我答

Q : 工作时间是怎样的，网上搜索到说公司工作时间很长，很少有节假日？

A : 要想获得不一样的梦想，必须要有不一样的付出。我们是一群有梦想的年轻人，一周六天的工作确实十分辛苦，但同时我们也一直在尽全力为大家营造舒适的工作环境，乐在其中的伙伴们并不认为这是个问题。公众的节假日我们均按国家的规定来放假，春节假期更会有多几天的时间以供大家与家人团聚。时不待我，想要进行创业的伙伴们，想要激情一把的伙伴们，加入我们！

Q : 毕业生的档案及户口怎么安排？

A : 毕业生可选择将档案及户口转到广州，我们与当地的南方人才市场有合作，迁往广州的档案及户口落在南方人才市场的集体户中。需要户口转移的同学要在签协议时提出，并将签好的协议及其它资料一并交往校方办理户口及档案转移手续。

Q : 实习时间是什么时候，实习期间待遇如何？

A : 实习时间根据每位同学的课程安排而异，实习时间当然越长越好，熟悉公司的同时也为未来的入职打好坚实的基础。我们以往有同学从11月份开始来公司实习直到毕业，毕业后直接转为正式员工。 实习期间，我们报销一次来往公司和学校的火车硬卧车票，包吃包住，另外每月给3000元的生活补助。需要提醒的是，实习依然是面试环节的一部分，通过实习考核才能获得我们正式OFFER。

Q : 实习时间是什么时候，实习期间待遇如何？

A : 实习时间根据每位同学的课程安排而异，实习时间当然越长越好，熟悉公司的同时也为未来的入职打好坚实的基础。我们以往有同学从11月份开始来公司实习直到毕业，毕业后直接转为正式员工。 实习期间，我们报销一次来往公司和学校的火车硬卧车票，包吃包住，另外每月给3000元的生活补助。 针对2020届毕业生，通过实习才能获得我们正式OFFER。

Q : 在朗国的职业发展是怎样的？

A : 个人的发展根据个人的能力与特长。有的同学对开发充满热情，我们可以将开发进行到底，从工程师，到项目工程师，再到高级工程师；有的同学对管理见解独到，在累积了一两年的经验后可发展为部门主管、项目经理、部门经理。转正满三年，符合公司价格观的员工，更有晋升为团队股东的机会。人的潜力是无限的，朗国的宗旨是不断的给优秀的员工提供广阔平台。
